Artificial intelligence aims to create machines capable of intelligent behavior. Machine learning enables systems to learn from data without explicit programming. Deep learning uses neural networks with many layers. supervised learning trains on labeled examples; unsupervised learning finds patterns in unlabeled data; reinforcement learning learns through trial and error with rewards. Natural language processing allows computers to understand and generate human language. Computer vision interprets visual information. Knowledge graphs represent relationships between entities. Search algorithms like A* find optimal paths. Logic and reasoning systems draw inferences from facts. AI ethics considers bias, fairness, and societal impact. The Turing test evaluates whether a machine can exhibit human-like intelligence. Current AI systems are narrow, excelling at specific tasks but lacking general intelligence.
